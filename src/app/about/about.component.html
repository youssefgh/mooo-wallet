Mooo wallet

Open Source Segwit wallet for Bitcoin.

No private key is sent over the network ! The transaction signing is done on your computer.

We don't save your keys so make sure to backup them safely and use them only on virus free computers for maximum security.

Main futures :

  • Creation and sending from Native Segwit addresses also known as Bech32 addresses
  • Creation and sending from Segwit compatible addresses also known as P2SH(P2WPKH) addresses
  • Sending to multiple addresses in the same transaction to save on transaction fees

Use at your own risk.

Check the source code, report issues and contribute to this project via Github
